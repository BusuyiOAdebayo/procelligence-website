Procelligence AI Website Chatbot

Files:
- index.html
- style.css
- api/chat.js

Vercel environment variables:
- OPENAI_API_KEY = your OpenAI API key
- OPENAI_MODEL = gpt-4.1-mini (optional)

After adding the environment variable, redeploy the Vercel project.
Do not place the API key in index.html, style.css, or api/chat.js.
