# Procelligence AI Website

Responsive static website centered on **Agentic AI for Computational Science & Engineering Intelligence**.

## Run locally
Open `index.html` in a browser, or run:

```bash
python -m http.server 8000
```

## Deploy
Replace the corresponding files in your GitHub/Vercel repository, commit, and push.

## Before publishing
- Confirm the public contact email.
- Add the final company logo and LinkedIn URL.
- Review all wording for legal and employment-policy alignment.
